## 📖 第一部分：前情提要與設計理念

### 🎯 KCISLK ESID 計畫願景

**KCISLK ESID** (Kang Chiao International School Linkou - Elementary School International Department) 是一套專為林口康橋國際學校小學國際部設計的數位化教學管理平台。

我們的願景是打造一個整合、高效、直覺的工作環境，讓教職員可以：

- **一站式存取** 所有教學工具和資源
- **無縫銜接** 行政管理與教學工作
- **即時掌握** 學生學習狀況與成績數據

### 🔗 兩個核心系統

KCISLK ESID 由兩個緊密整合的系統組成：

#### Info Hub (TeacherOS) — 您的教學工作站

**入口網址**: `https://kcislk-infohub.zeabur.app`

Info Hub 是您每天開啟工作的起點，採用 **macOS 風格設計**，提供：

- 📅 **行事曆 Widget** — 學校重要日期一目了然
- 📝 **個人待辦 Kanban** — 拖曳式任務管理
- 📨 **公告訊息板** — 同步 Office 公告（Google Doc 嵌入）
- 📁 **重要文件夾** — Finder 風格的文件快速存取
- 🚀 **Launchpad** — 36 個教學 App 的快捷入口
- 🎓 **LMS 快速連結** — 一鍵進入成績管理系統

#### LMS (Learning Management System) — 成績與課程管理

**入口方式**: 透過 Info Hub 的 🎓 LMS 圖示，**無需另外登入**（SSO 單一登入）

LMS 是專業的學習管理系統，提供：

- 📊 **成績輸入與計算** — 智慧化成績管理
- 📈 **進度追蹤** — 班級成績輸入進度一覽
- 📋 **課程看板** — Kanban 式課程任務管理
- 📉 **數據分析** — 班級統計與學生表現分析

### 🏫 一班三師架構

KCISLK 小學國際部採用獨特的「**一班三師**」教學架構，每個班級有三種課程類型：

| 課程代碼 | 全名                    | 說明                 | 授課教師      |
| -------- | ----------------------- | -------------------- | ------------- |
| **LT**   | Local Teacher           | 本地英語語言藝術課程 | 本地教師      |
| **IT**   | International Teacher   | 國際英語語言藝術課程 | 國際教師      |
| **KCFS** | Kang Chiao Future Skill | 康橋未來技能課程     | KCFS 專任教師 |

每個學生會同時修習這三種課程，由不同專業背景的教師分別授課。

### 🔐 角色與權限設計

系統採用 **角色基礎存取控制 (RBAC)**，確保每位使用者只看到與自己職責相關的功能。

#### 您的角色決定您能做什麼

| 角色                       | Info Hub 功能      | LMS 功能                       |
| -------------------------- | ------------------ | ------------------------------ |
| **Head Teacher** 年段主任  | 教師專區全功能     | 監控年段進度、瀏覽成績（唯讀） |
| **Teacher** 教師           | 教師專區全功能     | 輸入/編輯自己班級成績          |
| **Office Member** 行政人員 | 公告管理、教師專區 | 全校唯讀、自己班級可編輯       |

### 📊 LMS 系統測試

#### 測試 1: 查看班級成績輸入進度

**這個功能幫助您**：一眼看出哪些班級進度正常、哪些落後。

**測試步驟**:

1. 從 TeacherOS 點擊 🎓 **LMS** 圖示進入 LMS
2. 點擊左側選單「**Browse**」→「**Gradebook**」
3. 查看班級列表

**您會看到**:

- 只有您指定年段的班級（如 G4 的 14 個班）
- 只有您指定課程類型的進度（如只有 LT）
- 每個班級旁有進度條

**進度計算邏輯**（供您驗證）:

```
進度 % = (已輸入成績數) ÷ (學生數 × 13) × 100

13 個評量項目 = FA1~FA8 (8個) + SA1~SA4 (4個) + MID (1個)
```

**計算範例**:

- 班級有 20 位學生
- 總共需要 = 20 × 13 = **260 筆成績**
- 教師已輸入 130 筆 → 進度 = 130 ÷ 260 = **50%**

**進度狀態顏色**:
| 進度 | 狀態 | 顏色 | 意義 |
|------|------|------|------|
| ≥ 80% | On Track | 🟢 綠色 | 進度正常 |
| 1% ~ 79% | Behind | 🟡 黃色 | 進度落後，需關注 |
| 0% | Not Started | ⚪ 灰色 | 尚未開始，需提醒 |

**✅ 預期結果**:

- [ ] 只看到自己年段的班級
- [ ] 只看到自己課程類型的進度
- [ ] 進度條顏色符合上述規則
- [ ] 可點擊進入任一班級查看詳情

#### 測試 3: 輸入成績並驗證計算

**測試步驟**:

1. 找到任一學生的空白成績格
2. 點擊格子並輸入以下測試分數：
   - FA1: **80**
   - FA2: **90**
   - SA1: **85**
   - MID: **75**
3. 按 **Tab** 移到下一格（或 **Enter** 確認）
4. 觀察右上角儲存狀態：
   - 「Saving...」（儲存中）
   - 「**✓ Saved**」（已儲存，綠勾）
5. 按 **F5** 重新整理頁面，確認分數仍在

**🧮 成績計算公式**（供驗證）:

```
學期成績 = (FA 平均 × 0.15 + SA 平均 × 0.20 + MID × 0.10) ÷ 0.45
```

**計算範例**:

```
Step 1: 計算 FA 平均
FA 平均 = (80 + 90) ÷ 2 = 85
（只有 FA1 和 FA2 有分數；空白的 FA3~FA8 不計入）

Step 2: 計算 SA 平均
SA 平均 = 85
（只有 SA1 有分數）

Step 3: 套用公式
學期成績 = (85 × 0.15 + 85 × 0.20 + 75 × 0.10) ÷ 0.45
         = (12.75 + 17 + 7.5) ÷ 0.45
         = 37.25 ÷ 0.45
         = 82.78
```

**⚠️ 重要規則**:
| 規則 | 說明 |
|------|------|
| 分數範圍 | 只允許 0 ~ 100 |
| 零分處理 | 0 **不計入**平均（視為未輸入） |
| 空白格 | 空白格不影響計算 |
| 全空/全零 | 平均顯示為「-」（無資料） |

**✅ 預期結果**:

- [ ] 成功輸入分數
- [ ] 右上角顯示「✓ Saved」
- [ ] 重新整理後分數仍在
- [ ] 如系統顯示學期總成績，與上述公式計算一致

#### 測試 8: MAP 成績分析（僅 G3-G6）

> ⚠️ **注意**：MAP 功能僅適用於 **G3 ~ G6** 年級。G1、G2 班級不會顯示此分頁。

**什麼是 MAP？**

NWEA MAP Growth 是美國標準化測評系統，用於追蹤學生的閱讀和語言能力成長。KCISLK 每年進行兩次 MAP 測試：

- **Fall** (秋季)：學年開始
- **Spring** (春季)：學年結束

**測試步驟**:

1. 進入一個 G3、G4、G5 或 G6 年級的班級
2. 點擊「**MAP**」分頁
3. 查看 MAP 相關數據

**MAP 頁面顯示內容**:

| 區塊               | 說明                                 |
| ------------------ | ------------------------------------ |
| **RIT 分數**       | 學生的 RIT（Rasch Unit）分數         |
| **Benchmark 分類** | E1（進階）/ E2（中級）/ E3（發展中） |
| **成長追蹤**       | Fall → Spring 的成長幅度             |
| **國家常模對比**   | 與美國全國平均值比較                 |

**MAP 課程範圍**:

- **Reading** — 閱讀理解
- **Language Usage** — 語言運用

**Benchmark 分類說明**:

| 等級   | 英文名稱     | 說明     | 用途                   |
| ------ | ------------ | -------- | ---------------------- |
| **E1** | Advanced     | 進階程度 | 下學年英語程度分班依據 |
| **E2** | Intermediate | 中級程度 | 下學年英語程度分班依據 |
| **E3** | Developing   | 發展中   | 下學年英語程度分班依據 |

**✅ 預期結果**:

- [ ] G3-G6 班級可看到 MAP 分頁
- [ ] G1-G2 班級**沒有** MAP 分頁（正確行為）
- [ ] 可查看學生 RIT 分數
- [ ] 可查看 Benchmark 分類
- [ ] 可查看成長數據（如有多期測試資料）

### 評量代碼完整說明

LMS 使用標準化的評量代碼系統：

| 類型           | 代碼      | 說明                     | 權重   |
| -------------- | --------- | ------------------------ | ------ |
| **形成性評量** | FA1 ~ FA8 | 平時評量（小考、作業等） | 共 15% |
| **總結性評量** | SA1 ~ SA4 | 單元評量（測驗、專題）   | 共 20% |
| **期中考**     | MID       | 期中測驗                 | 10%    |
| **期末考**     | FINAL     | 期末測驗                 | 10%    |

**成績計算公式**：

```
學期成績 = (FA 平均 × 0.15 + SA 平均 × 0.20 + MID 或 FINAL × 0.10) ÷ 0.45
```

**重要規則**：

- 只有大於 0 的分數計入平均
- 0 分視為「未輸入」，不影響計算
- 空白欄位不計入

## ❓ 第六部分：常見問題 FAQ

### Q1: 不同角色能看到什麼？

**A**: 請參考權限比較表。簡單來說：

- **Head Teacher**: 自己年段 + 課程類型
- **Teacher**: 自己被指派的班級
- **Office Member**: 全校（唯讀），自己班級可編輯

### Q2: 為什麼我無法編輯成績？

**A**: 您只能在以下情況編輯成績：

- 您是該班級和課程類型的**任課教師**
- 您是 **Office Member** 且該班級是您的授課任務

### Q3: 登入後看到空白頁面？

**A**: 按 F5 重新整理。如果仍然空白，請清除瀏覽器快取後重新登入。

### Q4: 從 TeacherOS 進入 LMS 需要再次登入嗎？

**A**: 不需要。系統使用 SSO（單一登入），會自動登入。如果發生需重新登入的情況，請回報！

### Q5: 成績計算好像有誤？

**A**: 請注意以下規則：

- 只有 **大於 0** 的分數才會計入平均
- 零分視為未輸入
- 公式：`(FA 平均 × 0.15 + SA 平均 × 0.20 + MID × 0.10) ÷ 0.45`

### Q6: 什麼是「一班三師」架構？

**A**: KCISLK 小學國際部每個班級有三種課程：

- **LT** (Local Teacher) — 本地教師英語課
- **IT** (International Teacher) — 國際教師英語課
- **KCFS** (Kang Chiao Future Skill) — 康橋未來技能課程

每位學生都會修習這三種課程，由不同教師授課。

### Q7: Head Teacher 和 Teacher 有什麼不同？

**A**:

- **Head Teacher**：監督整個年段的教學進度，可查看但**不能編輯**成績
- **Teacher**：負責自己班級，**可以編輯**成績
